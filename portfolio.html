Algoritmos em Python

Algoritmos feitos para o trabalho da disciplina Lógica de Programação e Algoritmos.

  • Vendas Parceladas

    Algoritmo para calcular o valor das parcelas e o valor total da compra com juros, se os requisitos forem atendidos.

    Veja no Google Colab

  • Loja de Marmitas

    Algoritmo para realizar o pedido de marmitas conforme os sabores e os tamanhos escolhidos e apresentar no final o valor total da compra.

    Veja no Google Colab

  • Fábrica de Camisetas

    Algoritmo para processar o pedido de camisetas conforme o tamanho, aplicando desconto sobre a quantidade de camisetas escolhidas, se os requisitos forem atendidos. Ao final, o sistema deve calcular o valor total, incluindo o frete, e exibir um resumo detalhado do pedido.

    Veja no Google Colab

  • Gerenciamento de Funcionários

    Algoritmo para gerenciar o cadastro de funcionários de uma empresa. O programa armazena nome, setor e salário em um dicionário, utilizando um ID global como ponto de partida. É possível consultar os funcionários por ID, por setor ou visualizar a lista completa. O sistema também inclui um menu para remover funcionários pelo ID.

    Veja no Google Colab
